================================================================================

PROBLEM: "pdflatex not found"
SOLUTION: Install LaTeX (see INSTALLATION.md)

PROBLEM: "Port 5000 already in use"
SOLUTION: Edit server.js, change port from 5000 to 3000

PROBLEM: "Cannot find module 'express'"
SOLUTION: Run npm install

PROBLEM: PDF won't display
SOLUTION: Check browser console (F12) and server logs
