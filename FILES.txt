================================================================================
                    FILE PURPOSES & RELATIONSHIPS
================================================================================

STARTUP FLOW:
1. package.json â†’ Defines npm start command
2. npm start â†’ Runs server.js
3. server.js â†’ Starts Express server on port 5000
4. Express â†’ Serves index.html
5. index.html â†’ Loads app.js and styles.css
6. app.js â†’ React app starts and renders UI

COMPILATION FLOW:
1. User types in app.js (React)
2. Auto-compile triggered after 1 second
3. app.js sends POST /api/compile
4. server.js receives request
5. server.js executes pdflatex
6. PDF generated and encoded to base64
7. Sent back to app.js
8. app.js renders with PDF.js
9. Preview updates in real-time

DOWNLOAD FLOW:
1. User clicks Download button in app.js
2. app.js initiates GET /api/download/:sessionId
3. server.js finds PDF file
4. Browser receives file download
5. PDF saved to user's downloads folder
