================================================================================

STEP 1: Install LaTeX
  
  Windows:
    - Download MiKTeX: https://miktex.org/download
    - Run installer and follow prompts
  
  macOS:
    - brew install basictex
  
  Linux (Ubuntu):
    - sudo apt-get install texlive-latex-base
    - sudo apt-get install texlive-latex-extra

STEP 2: Install Node.js
  - Download from https://nodejs.org/
  - Run installer
  - Verify: node --version && npm --version

STEP 3: Install Project Dependencies
  $ npm install

STEP 4: Start Server
  $ npm start
  
  Expected output:
  "Server running on http://localhost:5000"

STEP 5: Open in Browser
  - Go to http://localhost:5000
  - Start editing LaTeX!

================================================================================
                         API ENDPOINTS
================================================================================

POST /api/compile
  - Compiles LaTeX to PDF
  - Input: { latexCode: "..." }
  - Output: { pdf: "base64...", sessionId: "..." }

GET /api/download/:sessionId
  - Downloads compiled PDF
  - Returns: PDF file

GET /
  - Serves main HTML page
  - Returns: index.html

GET /styles.css
  - Serves CSS styling
  - Returns: styles.css

GET /app.js
  - Serves React application
  - Returns: app.js

================================================================================
                         PROJECT STRUCTURE
================================================================================

FRONTEND FLOW:
  User Types LaTeX ‚Üí Auto-compile timer ‚Üí Send to server ‚Üí Receive PDF ‚Üí 
  Decode base64 ‚Üí Render with PDF.js ‚Üí Display in preview

BACKEND FLOW:
  Receive LaTeX ‚Üí Write to temp file ‚Üí Run pdflatex ‚Üí Generate PDF ‚Üí 
  Encode to base64 ‚Üí Send to client ‚Üí Cleanup after 1 hour

COMPILATION:
  1. User stops typing
  2. 1-second debounce waits
  3. LaTeX code sent to server
  4. pdflatex executes compilation
  5. PDF encoded in base64
  6. Sent back to browser
  7. Rendered using PDF.js
  8. Displayed in real-time

BEFORE REINSTALLING:

1. Check if pdflatex is installed:
   $ pdflatex --version

2. Check if port 5000 is available:
   Linux/Mac: $ lsof -i :5000
   Windows:   netstat -ano | findstr :5000

3. Check if dependencies are installed:
   $ npm list

4. Clear browser cache (Ctrl+Shift+Delete)

5. Restart the server (Ctrl+C then npm start)
